# PlayingMusic

PlayingMusic adalah sebuah proyek Python yang dirancang untuk memutar lagu ataupun video yt lewat terminal. Dalam proyek ini, saya menggunakan **virtual environment** untuk mengelola dependensi, sehingga memastikan bahwa proyek berjalan dengan baik di berbagai lingkungan pengembangan.

## Prasyarat

Sebelum mulai, pastikan Anda memiliki hal-hal berikut:

- Python 3.x terinstal di sistem Anda.
- `pip` (Python Package Installer) untuk mengelola dependensi.

## Instalasi

Ikuti langkah-langkah di bawah ini untuk menyiapkan proyek ini di lingkungan pengembangan Anda.

### 1. Clone Repository

Pertama, clone repository ini ke mesin lokal Anda:

```bash
git clone https://github.com/username/PlayingMusic.git
cd PlayingMusic
```

### 2. Create virtual environment
```bash
python -m venv venv
```

### 3. Run using virtual environment
Using Fish :
   ``` bash
    source venv/bin/activate.fish
   ```

Using Bash/zsh :
  ``` bash
    source venv/bin/activate
  ```

### 4. Run PlayMusic
``` bash
python index.py
```
